### [Demo](http://ng2-webpack-starter.s3-website-us-east-1.amazonaws.com/)

To install dependencies and type definitions:

    npm install
    
To run tests with karma:

    npm test
    
To run the webpack development server:

    npm start
    
To trigger a build:

    npm run build
